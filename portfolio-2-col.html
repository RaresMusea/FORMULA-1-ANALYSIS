Formula One Pilots Analysis

Have you ever thought what it would be like if all Formula 1 drivers were just as good? Exactly! There would be no more competition! And because this competition is quite varied, I drew up, with the help of Python language, these graphics to illustrate certain details about the performance of the main attraction of these automotive events: drivers

Formula One-Driver Standings

Built through the Piktochart application, these interactive infographics complement the graphics developed with the help of the Python language. A repatriation of the most skilled pilots, depending on the countries and the number of victories.
