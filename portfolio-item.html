Project Description

This project aims to demonstrate that the visualization and analysis of data can be done in an intuitive and interactive way. The role of this project is a persuasive one. Users prefer to obtain information from environments that capture their attention and do not bore them. In the near future, everything will move to the web. More and more applications are already adopting this strategy and enjoying a resounding success. We want to prove that web programming languages, along with all related frameworks are the same, if not even more efficient than programming languages ​​designed to produce desktop applications.
